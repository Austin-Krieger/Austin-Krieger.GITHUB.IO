Adding Themes to Notepad++

  1. Open Notepad++ and click on Settings.
  2. Now open Style Configurator....
  3. You will see a dropdown box with the themes you currently have installed.
  4. In File Explorer type in %AppData%\Notepad++\themes
  5. Move your downloaded themes to this folder and restart Notepad++ to see the new themes.
  6. User Manual: https://npp-user-manual.org/docs/getting-started/
  7. Some Downloadable Themes: https://www.digitaldesignjournal.com/best-notepad-themes/
