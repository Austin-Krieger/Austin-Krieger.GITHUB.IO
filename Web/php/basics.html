PHP Basics

The PHP code (everything between <?php and ?>) will be interpreted by the Web server and converted to normal text before it's sent to your browser. The Web server does all the work, so the Web browser doesn't need to know anything about the php code.

Basic Syntax

A PHP script consists of a series of commands, or statments, each of which is an instruction that the Web server must follow before it can proceed to the next. PHP statements are always terminated by a semicolor (;).


echo("This is a <b>test</b>!");

This statement invokes a built-in function called echo.

Built-in functions are functions that PHP has pre-defined and do not require us to tell them how to work.

Quotes are used to mark the beginning and end of strings of text in PHP.

Parenthesis serve a dual purpose in PHP.

  1. They indicate that echo is a function that you want to call.
  2. They mark the beginning and end of a list of "parameters" you wish to provide.

Variables

Variables: A name given to an imaginary box into which any value may be placed.


$testvariable = 3;

PHP is a loosely typed language. This means that a single variable may contain any type of data (number, string, etc.) and may change types over its lifetime.


This line will assign a new value to our existing variable and change its type.
$testvariable = "Three";

Operators

The equals sign '=' is called the assignment operator, as it is used to assign values to variables.

Other operators may be used to perform various mathematical operations on values:


$testvariable = 1 + 1; // Assigns a value of 2.
$testvariable = 1 - 1; // Assigns a value of 0.
$testvariable = 2 * 2; // Assigns a value of 4.
$testvariable = 2 / 2; // Assigns a value of 1.

Comments

Comments are a way to describe what your code is doing.


// Tells the PHP interpreter to ignore what comes after.

Concatenation


$testvariable = "Hi " . "there!"; // Assigns a value of "Hi there!".

Variable Interpolation


$var1 = 5; // Assigns a value of 5 to $var1
$var2 = $var1 + 1; // Assigns a value of 6 to $var2

Control Statements

The most basic, and most often-used, control structure is the if-else statement.


if (condition) {
 // Statement(s) to be executed if condition is true.
} else {
 // (Optional) statement(s) to be executed if condition is false.
}

The double-equals (==) used in the condition section is the PHP operator that's used to compare two values to see whether they're equal. (Using a single equals sign would cause the condition to always be true.)

When checking two conditions use the keyword "and".


if ($firstname == "Thirsty" and $lastname == "Penguin") {
 // Do something

While Loop

The while loop allows us to use a condition to determine how many times to repeatedly execute a set of statements.


while (condition) {
 // statement(s) to execute as long as the condition remains true.
}

Loops such as this one come in handy when working with long lists.


$count = 1;
while ($count <= 10) {
 echo("$count");
 $count++;
}

  1. Line one creates a variable called $count and assigns it a value of 1.
  2. Line two is the start of the while loop, the condition is (Value of $count is less than or equal to 10).
  3. Line three and four are the body of the while loop, and will be executed over and over, as long as the condition holds true.
  4. Line three outputs the value of $count.
  5. Line four increments (adds 1 to) the value of $count.
    • Increment: $count = $count + 1, $count++
    • Decrement: $count = $count - 1, $count--
 

PHP Forms

As PHP code is activated when a page is requested from the server, user interaction can only occur in a back-and-forth fashion.

The user sends requests to the server, and the server replies with dynamically generated pages.

We can pass variables along with the page request. The variable is passed as part of the "query string", which is the portion of the URL that follows the question mark.

The variables firstname and lastname have the values Thirsty and Penguin respectively.


<a href="welcome.php?firstname=Thirsty&lastname=Penguin">Hi, I'm Thirsty, Penguin!</a>

We can accomplish the same using a form with the GET method.


<form action="welcome.php" method="get">
 <label for="firstname">First name:</label>
 <input type="text" name="firstname">
 <label for="lastname">Last name:</label>
 <input type="text" name="lastname">
 <input type="submit" value="GO">
</form>

In welcome.php we need to add a line that can print the data passed in the query string.


<?php
 echo("Welcome to my website, $_GET[firstname], $_GET[lastname]!");
?>

Using a form with the POST method will not pass information in the query string. However, this will allow you to include large values, or sensitive values (passwords) in the data that's submitted by the form.

If the user bookmarks the resulting page, it will be useless, as it doesn't contain the sumbitted values.


<form action="welcome.php" method="post">...</form>

In the welcome.php page we need to change $_GET to $_POST:


echo("Welcome to my website, $_POST[firstname], $_POST[lastname]!");

 

Download & Install MySQL

You can downlaod and install MySQL software from https://mysql.com. In windows the program is called mysql.exe and is located by default in the C:\mysql\bin.

If you end up at a command line use the following line: mysql -h hostname -u username -p (-p prompts for a password).

If the client program is run on the same computer as the server, you can leave off the -h hostname part (-h localhost, -h 127.0.0.1)

If you installed the MySQL server yourself, this will just be root.

 

SQL

SQL or Structured Query Language, pronunciation "sequel" or "ess-cue-ell". Commands in SQL are also called Queries.

UNIX-based systems will be case-sensitive when it comes to database and table names, as they correspond to directories and files in the MySQL data directory. Otherwise, MySQL is completely case-insensitive.

Table, column, and other names must be spelled exactly the same when they're used more than once in the same command.
