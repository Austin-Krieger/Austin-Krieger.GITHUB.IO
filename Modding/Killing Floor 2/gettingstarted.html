Installing SteamCMD

  1. Open the File Explorer to the Local Disk (C:) drive folder.
  2. Create a new folder named steamcmd.

    3. C:\steamcmd

  3. Drag the steamcmd.zip file from above into the new folder and extract the files using your prefered extraction software. 7-zip for example.
  4. Double-Click steamcmd.exe to open the Steam Console Client from Valve.
  5. Running steamcmd opens a command prompt window that will install the files necessary for running steamcmd and installing servers.

Logging In & Installing Server Files

  1. We want to tell steamcmd where to install the server right away. Try running the following command:

    2. Steam>force_install_dir ..\KF2Server\

  2. Some commands require you to login. Using your own account or even one that owns the game however is not required, so we will be logging in as anonymous which will give us access to everything we need.

    3. Steam>login anonymous

  3. Now we want to install the killing floor 2 dedicated server files. You can check the references section above for more details, but the command we want to run next will install and verify the integrity of the server files.
  4. Try running this command next. (It will double as our update command later)

    5. Steam>login anonymous +force_install_dir ./KF2Server +app_update 232130

  5. If that command did not start the download then move on to this next command or try adding validate to the end of the one we just used. Validate should make sure the server files are actually downloaded. Around 20GB of free space is required for the server files, so make sure you have room on your drive.

    6. Steam>app_update 232130 validate

Missing Dependencies

  1. For the next step we need to take care of some missing dependencies. Navigate to the Redist folder inside of the kf2server folder and run UE3Redist.exe. When that is done we can close the application.

    2. C:\steamcmd\steamapps\common\kf2server\Binaries\Redist

Server Configuration

At this point we have a couple options to consider when configuring our server. Option #1 is a little more automated and made easy with the use of a program called KF2ServerTool and Option #2 is more manual setup without any programs to add lines to .ini files for us. Both options should be able to download custom maps automatically from the workshop, so players don't need to download and subscribe to each and every map themselves.
(Some of these Steps are not necessarily shared between both options, but it wouldn't hurt to complete the following steps either way)

  1. Navigate to the Config folder and find the PCServer-KFGame.ini file.

    2. C:\steamcmd\steamapps\common\kf2server\KFGame\Config

  2. Search for the [Engine.AccessControl] section and add a new admin password.

    3. AdminPassword=A1b2C34D

  3. Search for the [Engine.GameReplicationInfo] section and set the server name to whatever you want, I went with BNHS Server because it is recognizable for my friends.

    4. ServerName=BNHS Server

  4. Back in the Config folder, find and edit the KFWeb.ini file.
  5. Search for the [IpDrv.WebServer] section and change bEnabled=false to true. This will allow us to connect to and modify server settings from a browser.
  6. Before we try to test our server, go back out to the kf2server folder and find the KF2Server.bat file.
  7. Right click the file and click edit. Use the example below and replace the admin name with the name of your choice and password with the password we set earlier in the tutorial. The difficulty count starts at 0=normal, 1=hard, etc. The port settings I have added to the second example are for making the server public and joinable by friends that don't share a LAN connection, so if you are only running it on your local network then you can use the first one.

    8. LAN:


    start .\Binaries\win64\kfserver kf-bioticslab?adminname=yourname?adminpassword=A1b2C34D?Difficulty=1?GameLength=1

    Public:


    start .\Binaries\win64\kfserver kf-bioticslab?adminname=yourname?adminpassword=A1b2C34D?Difficulty=1?GameLength=1 -Port=7777 -QueryPort=27015 -WebAdminPort=8080

  8. Mutators: If you have mutators downloaded as well then you should be able to get the necessary line from the developer on the workshop page. Mutators will likely disable experience gain, but if you still want them or found one that does not disable experience, then here is an example of the line used for Friendly Hud.

    9. start .\Binaries\win64\kfserver kf-bioticslab?adminname=yourname?adminpassword=A1b2C34D?Difficulty=1?GameLength=1?mutator=FriendlyHUD.FriendlyHUDMutator -Port=7777 -QueryPort=27015 -WebAdminPort=8080

  9. The KF2ServerTool has a MOD tab that should let you add workshop mutators to the server pretty easily, but if you are setting up the server manually then you should just need to add another ServerSubscribedWorkshopItems= line for the mutator workshop item.
  10. At this point in option #2 you could try starting up the server by double clicking on the KF2Server.bat file, but setting up option #1 will just takes a few more steps, and they are well worth the effort. We will add workshop maps to our server and learn how to change settings in Web Admin next.

Option #1 - KF2ServerTool

  1. For this option we will be using the KF2ServerTool. Download the latest release and move the .zip file to our C:\steamcmd\kf2server folder. The same one that contains KF2Server.bat
  2. Unzip the files and run KF2ServerTool.exe.
  3. Once open you will see somthing similar to the image below. Create a new profile with your server name, add in the port settings (if your are making it public) and move on the the other settings tabs.
  4. From the Options tab, check that Workshop Download Manager is enabled.
  5. Moving on to the Maps tab we can start adding workshop maps for players to automatically download.
  6. Click on the big +Add button in the bottom left and select From Workshop Browser. Type in the name of the workshop map you wish to download and a window will appear with the workshop options that match what you typed in. Subscribe your server to the map and you will see a new section appear in the maps list for Steam Workshop maps. Within this process you can also add the map to the Map Cycle which mean it will appear in the map vote options after each match inside of the game and within the WebAdmin tab.
  7. If you have issues running your server or with the maps not downloading, you can always try right-clicking on the map names and forcing an update, reinstalling, and/or removing it from cache. It may be best to just remove the map completely and try doing the add from workshop step again for each workshop map.
  8. With that out of the way you can go back to the Server tab and hit Start Server. This will open a similar batch process to option #2, but you no longer need to go to a web browser to change settings. Just head over to the WebAdmin tab where you will be able to change the map, make a custom map cycle, add mutators, change the clan motto and so on.
  9. When changing the map, you may want to have the port settings on hand just in case. Paste them into the Additional URL Variables section like in the example below and hit the change map button. Port settings are discussed more in the Making The Server Public section below.
  10. If you have already tried testing your server, then your friends likely have been unable to join thus far, but don't worry, just head down to the Making The Server Public section below and configure your router to have port forwarding and your server computer to have a Static IPAddress. Once that is complete you can enjoy hosting your own KF2 Dedicated Server with all the custom maps you've been wanting to try out.

Option #2 - Manual Setup

  1. Continuing from the shared server configuration settings listed above we will work on adding maps to our server without the use of the KF2ServerTool.
  2. Before we can download our workshop maps, we need to get Steams' app id for Killing Floor 2. Go to SteamDB and search Killing Floor 2 and you will see App ID: 232090. Copy this number to a sticky note or txt file.
  3. Next we need the id of the actual workshop map. You can get this from inside the steam client or in your browser. From the Steam client you can navigate to the workshop map where the subscribe button is located and right click to copy the page url. You can also view the url in your browser and by changing the Steam settings to show the url inside of the Steam client. However you get there is not important. At the end of the url you just copied or in the address bar we need to find ?id= and a big number. Copy this number to a sticky note or txt file as well.


    4. https://steamcommunity.com/sharedfiles/filedetails/?id=643383080

  4. If you already closed the steamcmd command prompt from earlier then run the executable and log in as anonymous again. Type in the command workshop_download_item app-id workshop-id. replace app-id with the Killing Floor 2 app id we copied to our sticky note and then replace the workshop-id with the id of the map you want to download.


    5. Steam>workshop_download_item 232090 643383080

  5. You should now have the new map downloaded to the content > 232090 folder. Open this folder and the folder with the mod id and so on until you get to the .kfm file. Copy this file and then go to the BrewedPC\Maps folder and create a new folder called CustomMaps. Paste the file here for nice organization or in the Maps folder at the bottom of the list if you want.

    6. C:\steamcmd\steamapps\common\kf2server\KFGame\BrewedPC\Maps

  6. Back in the PCServer-KFGame.ini file we need to add the name of the map we just added to our server if we want to see the map from inside of Web Admin. Scroll down to the KF-Default map settings and add the new map to the list. Here is one example using the KF-BikiniAtoll map:

    7. [KF-BikiniAtoll KFMapSummary]
    MapName=KF-BikiniAtoll
    MapAssociation=0
    ScreenshotPathName=UI_MapPreview_TEX.UI_MapPreview_Placeholder

  7. The ScreenshotPathName and MapAssociation lines are not actually required and I would actually just recommend removing them. The example below shows both lines in use, but they are not required and really serve no purpose for custom maps.
  8. You are now ready to move on to the Web Admin section, which makes some things like changing game mode and map a bit easier than shutting down the server and changing them manually.

Accessing Our Server From a Web Browser

  1. You will need the IP address of the computer you are using as a server. Hit the windows key, type cmd, and hit enter to open a command prompt. Type in the command ipconfig and hit enter. You will need the IPv4 address: 192.168.1.?the last number was 16 in my instance.
  2. Now you will type that address into the address bar of your browser with :8080 at the end.

    3. 192.168.1.16:8080

  3. You will be asked to login, so use the name and admin password we set up previously in our tutorial.
  4. Change Map
  5. Change Map Queue

Making The Server Public

  1. In order to make the server public we need to make sure the KF2Server.bat file includes the port options shown earlier in the tutorial. Then comes the hard part.
  2. Remember the IP address of the computer you are using to host the server? We need to give it a Static IP Adress on our Router. If you are coming to this section from the Option #1 settings then you can scroll back up to the beginning of the Accessing Our Server From a Web Browser section which describes how to get your computers' IPAddress through the use of IPConfig.
  3. In a Browser type into the address bar: 192.168.1.1 this is a standard address for accessing your router. Login and search for your Static DHCP Configuration options and Port Forwarding options.
  4. In your Static DHCP Configuration options choose the device info that matches your device and IP address, make it active and hit OK.
  5. Next find your Port Forwarding options and start adding the ports we want open. refer to the example below for range values and names.

    6. Game Port: 7777 - 7778 | UDP
    Query Port: 27010 - 27019 | UDP
    Web Admin: 8080 - 8081 | TCP
    Steam Port: 20560 | UDP

  6. That should be everything for Option #1 to work properly and act as a public server.

Installing Custom Maps (Workshop)

  1. Make sure the server is not running before making your changes.
  2. We need to add a new section to the PCServer-KFEngine.ini config file. I would recommend adding this section above the [IniVersion] section.

    3. [OnlineSubsystemSteamworks.KFWorkshopSteamworks]

  3. For the next few lines we need to know the Workshop IDs for our subscribed workshop items and a comment at the end with the name of the map which will help us remember what workshop id matches with what map. If you were following the Option #1 steps earlier, these lines are actually added automatically by the KF2ServerTool program we downloaded and you would not need to mess with these settings personally.

    4. [OnlineSubsystemSteamworks.KFWorkshopSteamworks]
    ServerSubscribedWorkshopItems=1158143366 // NachtDerUntoten
    ServerSubscribedWorkshopItems=1130264785 // KF-DeepinWall
    ServerSubscribedWorkshopItems=1285074158 // KF-Arid_Zedlands
    ServerSubscribedWorkshopItems=643383080 // KF-BikiniAtoll

  4. Add more "ServerSubscribedWorkshopItems=" lines with the Workshop IDs of all the items you want your server to download. Also, you will still need to add the map names in your PCServer-KFGame.ini file in order to see them from inside of Web Admin.
  5. The steam workshop option will take precedence over the redirect option below, but they can still be used together. I included both options in the screenshot above, but I would actually recommend leaving the RedirectToURL= option blank like it was.

Installing Custom Maps (Redirect)

  1. Within the Config folder find the file named PCServer-KFEngine.ini, open this file and search for the [IpDrv.HTTPDownload] section. We will be adding onto the line: RedirectToURL=
  2. A popular yet I belive broken site that is used here is: https://www.skillzservers.com/kf2-redirect/

    3. [IpDrv.HTTPDownload] RedirectToURL=https://www.skillzservers.com/kf2-redirect/

  3. I have found another redirect site for some maps here: http://kf2.tripwirecdn.com/, but I still don't see this as the best option.
  4. The next step is to go to the link/web address yourself, download the map files you want, and place them in your KF2Server\KFGame\BrewedPC\Maps\CustomMaps folder that we made earlier. This way your server and everyone joining it will have the same version of the custom map.

Creating a Server Update Shortcut

  1. This shortcut can be used whenever the game gets an update, it is used to make sure our server files are all downloaded and up to date. For Option #1 users above, the KF2ServerTool comes with update buttons to make this process easier.
  2. We can setup a more automated update command for our server by right clicking on steamcmd.exe > Create shortcut > Rename the shortcut Update Server and then right click on our new shortcut and select properties.
  3. We want to use the following line in the target box and hit apply. If you are using a different drive or steamcmd is in a folder that is not named steamcmd then make your changes to the command below before hitting apply.

    4. C:\steamcmd\steamcmd.exe +login anonymous +force_install_dir ./KF2Server +app_update 232130 +exit

  4. Double click on the shortcut to run this command whenever you feel like an update is required. The command itself should login as anonymous, update the server (if necessary), and then exit the window.
  5. If you are experiencing any issues that can not be resolved by reading this guide then feel free to check out the references section at the top for more information and assisstance from other server hosts.
