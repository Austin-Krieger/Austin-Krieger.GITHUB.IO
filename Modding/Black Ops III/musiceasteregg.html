Music Easter Egg

This tutorial includes two options for adding a music easter egg to our map.

  1. Here are the folders you will need to open for this tutorial, starting with our root folder D:\Steam\steamapps\common\Call of Duty Black Ops III\

    2. usermaps\zm_tutorial_level\sound\zoneconfig

    usermaps\zm_tutorial_level\scripts\zm

    share\raw\sound\aliases

  2. Option #1 - Shoot Teddy Bears: Music_Easter_Egg_1.zip
  3. Option #2 - Touch Teddy Bears: Music_Easter_Egg_2.zip
  4. Once you extract the files, open up the Music_Easter_Egg_# folder and drag the contents into the root folder. You can also move the actual files into their appropriate folders yourself.
  5. Go to your Maps' sound\zoneconfig folder and open up the zm_tutorial_level.szc file.
  6. Add in the new alias below user_aliases as shown here:

    7. {
    "Type" : "ALIAS",
    "Name" : "user_aliases",
    "Filename" : "user_aliases.csv",
    "Specs" : [ ]
    },
    {
    "Type" : "ALIAS",
    "Name" : "zmb_music_ee",
    "Filename" : "zmb_music_ee.csv",
    "Specs" : [ ]
    },

  7. Next go to your Maps' scripts\zm folder and open the zm_tutorial_level.gsc file.
  8. Inside the main() function add the line that goes with the option you chose:

    9. // Music Easter Egg - Option #1
    level thread soundEasterEggInit();
    // Music Easter Egg - Option #2
    level thread EE_Music();

  9. Scroll down to the bottom of the file and paste the code from the Code.txt file.
  10. Now it's time to open our map in Radiant.
  11. Right Click > Misc > prefab. Double Click the Music_Easter_Egg folder and open the Prefab/s.
  12. For Option #1 you need to place all 3 prefabs or make a copy and change the number in the triggers' targetname.
  13. For Option #2 you can place the prefab and make multiple copies in the map; it can be more than 3.

Additional Reading

  1. When changing the song for this easter egg, make sure it's a .wav file with a Rate of 48000 Hz and has the appropriate name for the option you chose. Option #1: song_easter_egg1, Option #2: ee_music
  2. You can always change these names and modify the zmb_music_ee.csv file and your Maps' .gsc file.
  3. You can also change the model from the teddy bear to whatever you would like to use. Just Stamp the prefab and drag in a model from the Models Pane (M). You could also edit the prefab if you want to change it that way and make your own prefabs in the process.
  4. The Teddy Bear activation and looping sounds were created from some gameplay footage I took on the Ascension map.
  5. The Music for the easter egg is provided by Bensound.com (Royalty Free).
