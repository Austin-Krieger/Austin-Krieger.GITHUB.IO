Adding Perk Machines

Juggernog, Staminup...

Most of the default perk machines are simple to add (Juggernog, Quick Revive, Speed Cola, Deadshot, Double Tap, and Staminup)

  1. Right Click in the Game or Grid view and go to Misc > prefab. Go into the zm and zm_core folders and search for the perk you want to add.
  2. For example: vending_deadshot_struct.map
  3. This is one of the few perks that doesn't seem to come with a model you can view within mod tools, but it will show up in game.
  4. Thanks to the additional mod tools assets, the other perk machines will have models that appear in your Grid and Game views.

Widows Wine

You have a couple options for adding Widows Wine to your map, using the prefab provided or making your own.

  1. Download our Widows Wine prefab here: vending_widowswine_struct.map

  2. Move the file to your zm_core folder:

    3. D:\Steam\steamapps\common\Call of Duty Black Ops III\map_source\_prefabs\zm\zm_core

  3. Right Click in your map and go to Misc > prefab.
  4. Double Click the zm and zm_core folders and search for vending_widowswine_struct.map.
  5. Click Open

Use Juggernog to Make Widows Wine

  1. You can also make your own widows wine perk machine by adding a copy of the juggernog perk machine to your map and modifying the prefab.
  2. With perk machine selected, go to the Entity Info Pane (N) and change the targetname to p7_zm_vending_widows_wine.

    3. targetname: p7_zm_vending_widows_wine

  3. Change the script_noteworthy to specialty_widowswine

    4. script_noteworthy: specialty_widowswine

  4. Finally, go to File > Save As and name the new prefab vending_widowswine_struct.map. You don't want to save over your version of the juggernog machine.
  5. Now you can leave the copy of your juggernog prefab without saving over it and place your new widows wine prefab onto the map.

Adding Loose Change to Perk Machines

We can add loose change to our perk machine prefabs fairly easily by following these steps.

  1. Open your Map folder and go to the scripts\zm folder. Look for the file named zm_tutorial_level.gsc

    2. D:\Steam\steamapps\common\Call of Duty Black Ops III\usermaps\zm_tutorial_level\scripts\zm

  2. Right Click the file and edit it with your preferred text editor.
  3. Add the following line near the top of the Main() function:

    4. level zm_perks::spare_change();


    function main()
    {
     zm_usermap::main();

     // Loose Change
     level zm_perks::spare_change();

     level._zombie_custom_add_weapons =&custom_add_weapons;

     // Setup the levels Zombie Zone Volumes
     level.zones = [];
     level.zone_manager_init_func =&usermap_test_zone_init;
     init_zones[0] = "start_zone";
     level thread zm_zonemgr::manage_zones( init_zones );

     level.pathdist_type = PATHDIST_ORIGINAL;
    }

  4. Add the following lines to the Perks section:

    5. #using scripts\zm\_zm_perks;
    #insert scripts\zm\_zm_perks.gsh;


    // Perks
    #using scripts\zm\_zm_pack_a_punch;
    #using scripts\zm\_zm_perks;
    #insert scripts\zm\_zm_perks.gsh;
    #using scripts\zm\_zm_pack_a_punch_util;
    #using scripts\zm\_zm_perk_additionalprimaryweapon;
    #using scripts\zm\_zm_perk_doubletap2;
    #using scripts\zm\_zm_perk_deadshot;
    #using scripts\zm\_zm_perk_juggernaut;
    #using scripts\zm\_zm_perk_quick_revive;
    #using scripts\zm\_zm_perk_sleight_of_hand;
    #using scripts\zm\_zm_perk_staminup;
    #using scripts\zm\_zm_perk_widows_wine;
    #using scripts\zm\_zm_perk_electric_cherry;

  5. Now we need to edit some perk prefabs in Radiant.
  6. In the Entity Browser Pane find Trigger and Right Click > Toggle Favorite.
  7. Select a perk machine, I chose Electric Cherry
  8. Right Click > Prefab > Enter Prefab.
  9. We can now Right Click in the Game or Grid view and add a Trigger > once to the prefab. Other tutorials have you use a trigger multiple, but that just gives you either +200/+100+100/+100 (Not consistent or what we want).
  10. Resize and move the trigger box in front of the machine like so:
  11. With our Trigger selected, go to the Entity Info Pane (N) and change the targetname to audio_bump_trigger.

    12. targetname: audio_bump_trigger

  12. Click Add KVP

    13. Property / Key: script_sound
    Value: zmb_perks_bump_bottle

  13. Next you can copy the trigger box and paste it into your other perk machine prefabs.
  14. You can also download these prefabs with loose change trigger boxes included:
    15. vending_additionalprimaryweapon_struct.map
    vending_deadshot_struct.map

    2. D:\Steam\steamapps\common\Call of Duty Black Ops III\usermaps\zm_tutorial_level\scripts\zm

  2. Right Click the file and edit it with your preferred text editor.
  3. Add the following line near the top of the Main() function:

    4. level.perk_purchase_limit = 20;

     zm_usermap::main();

     // Perk Limit
     level.perk_purchase_limit = 20;

     level._zombie_custom_add_weapons =&custom_add_weapons;
