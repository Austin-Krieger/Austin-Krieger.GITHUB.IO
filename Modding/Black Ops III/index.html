Changing Views

These are the shortcuts for changing what you can see in the Game & Grid Views.

Action Control
Flat View F7
Lighting View F8
Game View F9
Cycle Views in Grid View Ctrl + Tab
 

Selection Shortcuts

Shortcuts for selecting, de-selecting, hiding, deleting, and duplicating.

Action Control
Select Everything i
De-select Everything Esc
Duplicate Selected Item Space
Delete Items Backspace
Hide Selection H
Show the Last Items Hidden Ctrl + Shift + H
Show All Hidden Items Shift + h
Select Rotation Handles R
Toggle Edge Lines for Selection J
Cut (Removes Portion of the Selected Item) X, Select 2 Points in Grid View, Enter
Cut (Keep Both Parts as Separate Objects) X, Select 2 Points in Grid View, Shift + Enter
