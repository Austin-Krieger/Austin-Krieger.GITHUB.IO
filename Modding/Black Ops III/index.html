Install Mod Tools

In the tools section of Steam find Call of Duty: Black Ops III - Mod Tools and install it.

It's also good to download the additional models and textures.

  1. Right Click the mod tools and click properties.
  2. Click on the DLC section at the bottom of the list. It should look like this:

Open Radiant Level Editor

  1. Lauch Call of Duty: Black Ops III - Mod Tools in Steam.
  2. Create a new map by clicking on the new icon or going to file > new.
  3. Make a name for our new zombies mod level starting with zm and splitting spaces with underscores. For example: zm_tutorial_level.
  4. Change the template to ZM Mod Level and hit OK.
  5. Hit OK again.
  6. The new map will appear in the Maps list on the left.
  7. The next step is to open the map in the Radiant Level Editor. Select your map from the list (Checking the Checkbox is Optional at this point) and go to File > Open in Radiant. You can also hit the Radiant button (Make sure the map is selected in the list).
  8. When Radiant opens it should look something like this:
  9. To rearrange the layout of the workspace, click and drag and refer to the Workspace Shortcuts table above.
  10. Add the Textures Pane by pressing T and drag it to the left of the Entity Browser Pane. You can exit out of the Entity Info Pane for now (Press N if you need to bring it back).
  11. Adding script to your favorites can also be useful later. Inside the Entity Browser Pane Right Click script and hit Toggle Favorite.

Making a Wall

  1. Add the Textures Pane by going to View > Toggle > Textures Pane or by pressing T.
  2. With the Textures Pane open click in the search bar and type caulk.
  3. Right Click and Add to Favorites.
  4. Above the search bar is an option called Favorites that when checked will filter to show only your favorited textures.
  5. The next texture to search for is clip. Right Click and Add to Favorites. This will be useful later.
  6. Click and Drag in the Grid View to make the wall.
  7. Clicking just outside the border lines of your new wall will allow you to change the size of the wall.
  8. To move your wall around, click and drag in the Grid View or use the Axis Handles in the Camera Pane.
  9. Remember you can change your view in the Grid View by hitting Ctrl + Tab. This is the best way to move your walls around to keep them in line with the floors and not floating. Scroll Wheel will allow you to zoom in and out in the Grid View as well.

Texturing Walls

  1. We can start by select a single face of our new wall (Shift + Ctrl + Click).
  2. Open the Texture Pane if it's not already open (T).
  3. Search for brick and Double-Click one of the options that appear.
  4. In the Textures Pane we can also filter our textures by selecting All or All in Use (Textures currently in use in the map) then select the Surface Type drop down and select brick.
  5. Pressing F9 in the Camera Pane will allow you to view your changes in Game View. We only applied the texture to this side of our wall so the rest of the wall will look a little bland. Press F9 again to exit Game View and press F7 to return to Flat View.

Copy Textures From Wall → Wall

  1. We can texture a wall with an existing wall texture. Select the wall face you want to texture using Shift + Ctrl + Click, hover over the wall texture you want to apply and Middle Mouse Click.
 

Adding Wall Buys

  1. In the Entity Browser find Misc, Right Click, and Toggle Favorite.
  2. Right Click in the Game View and hover over Misc > prefab. Without favoriting Misc this option won't appear.
  3. A new window will pop up, double click zm > zm_core.
  4. Search for spawnable_weapon_bowie.map and hit Open to add the Bowie Knife to the map.
  5. Move the prefab up against the wall and if it's backwards like in the image below, press R to switch to rotation handles and use the Z axis to flip it around.
  6. We can change the rotation manually from the Entity Info Pane. This is a better way to make sure our item is rotated 180°.
  7. When moving the prefab around you will notice that it snaps to the corners of the grid squares. We need to increase the number of grid squares to move it more precisely. Find the button that looks like a grid and choosing Grid 8.
  8. We need to line up the prefab with the floor & wall as shown below. Remember Ctrl + Tab will let you change the current Grid View. We want the wall buy to be very slightly off the wall, so if you switch the grid size to Grid 1 we can add a tiny square of space between the knife and the wall.
  9. The Bowie Knife should now be added and look similar to this in the Game View.

Change the Skybox

We can change the skybox from a daytime to a nighttime sky.

  1. Right Click in the top bar of the level editor and open the Entity List Pane (Lists all the item currently in the level).
  2. Find volume_sun in the list.
  3. Check the box next to volume_sun to select it.
  4. Press N to open the Entity Info Pane with the volume_sun properties.
  5. Find Light State 1 and click on default_day an elipses button will appear. Press it to open a list of skybox options.
  6. Search for and select zm_factory.
  7. Hit OK.
  8. In the Camera Pane switch to Lighting View (F8).
  9. You will need to Right Click next to the FX button and click Exposure to bring up more options.
  10. Click the drop down next to the camera icon on the right and click on the Lightning Bolt.
  11. We can now switch to Game View (F9) and then Lighting View (F8) to see our cooked lighting and pressing F9 again will exit Game View. F7 returns you to Flat View.
 

Setup Zombie Spawner

  1. In the Entity Browser Pane search for spawner_zm_factory.
  2. Click and drag it into the map. If you have the extra assets dlc installed, you will see the model of a zombie.
  3. With the entity selected (Shift + Click) go to the Entity Info Pane.
  4. First we need to change the coop_count from 1 to 9999.
  5. Change ALTERONSPAWN to False.
  6. Change MAKEROOM to True.
  7. Next we need to hit Add KVP to add some User-Defined properties.
  8. Click Add KVP

    9. Property / Key: script_disable_bleeder
    Value: 1

  9. Click Add KVP

    10. Property / Key: script_forcespawn
    Value: 1

  10. Click Add KVP

    11. Property / Key: script_noteworthy
    Value: zombie_spawner

Spawn Zombies (Map)

Now we need to add locations for the zombies to spawn.

  1. Right Click in the Game or Grid view and select script → struct.
  2. In the Entity Info Pane change the targetname to the name of the current zone. start_zone_spawners in this case.
  3. Click Add KVP.

    4. Property / Key: script_noteworthy
    Value: riser_location

  4. Click Add KVP.

    5. Property / Key: script_string
    Value: find_flesh

  5. You can use Space Bar to copy the selected struct and move it.

Double Doors

Description.

  1. In the Models Pane (M) search for p7_door_metal_industrial_02_left_white and drag it onto either the Map or Grid view.
  2. Do the same with p7_door_metal_industrial_02_right_white.
  3. Line them up with the floor and each other, like so:
  4. Next we need to select clip from the Textures Pane and make a clip the size of the doors themselves so that players can't walk through them.
  5. We still need a trigger box in front of the doors so the player will get a prompt to purchase it.
  6. Option #1 is to make another clip in front of the doors. Go to the Entity Browser and look for Trigger > use and with our new clip still selected, Double Click use.
  7. Option #2 is to still find Trigger > use, but to drag it onto the Grid or Game view and resize it the same way you would the clip from Option #1.
  8. Next we need to convert our clip to a brushmodel.
  9. Select our clip and return to the Entity Browser. Look for Script > brushmodel and Double Click.
  10. Select Door (Left) and this time find Script > model in the Entity Browser, and Double Click.
  11. Do the same for Door (Right).
  12. Next we can modify the Entity Info and add some KVPs.
  13. Select the Trigger.
  14. Scroll down to targetname. Change this to zombie_door, make sure this is spelled correctly or you will run into errors and your door will not work.
  15. We also need to set the cost of our door. Hit Add KVP and set the price of the door (Default is 1000).

    16. Property / Key: zombie_cost
    Value: 750

  16. Next we need to select the clip, but that may be a struggle will our trigger in the way. Select the trigger and press H to hide it. Hide anything else you want out of the way and when your done, press Shift + H to show everything again.
  17. With our clip selected, change the targetname to test_door.
  18. Click Add KVP

    19. Property / Key: script_noteworthy
    Value: clip

  19. Select the Door on the right. Note: this is a view from the opposite side of the doors from the trigger (Easier to select without hiding multiple entities).
  20. Change the targetname to test_door.
  21. Click Add KVP

    22. Property / Key: script_string
    Value: rotate

  22. Rotate the door until you find a rotation that fits your needs. In this case 0, 150, 0 looked good.
  23. Click Add KVP

    24. Property / Key: script_angles
    Value: 0 150 0

  24. Follow the same steps for the left door only this time use 240 in the rotation value.

    25. Property / Key: script_angles
    Value: 0 240 0

  25. Select our trigger entity one more time.
  26. Click Add KVP

    27. Property / Key: target
    Value: test_door

  27. Click Add KVP (This one is used when you have multiple zones)

    28. Property / Key: script_flag
    Value: enter_test_zone

Change the Starting Weapon

This is meant to change your starting pistol when loading into the map.

  1. Open your usermaps folder by going to your Steam Black Ops III folder. For example: D:\Steam\steamapps\common\Call of Duty Black Ops III\usermaps
  2. We need to find zm_tutorial_level\scripts\zm folder and you will see zm_tutorial_level.gsc
  3. Right Click the file and edit it with your prefered text editor.
  4. We need to add a few lines of code to the bottom of the main() function as shown here:

    5. function main()
    {
     zm_usermap::main();

     level._zombie_custom_add_weapons =&custom_add_weapons;

     //Setup the levels Zombie Zone Volumes
     level.zones = [];
     level.zone_manager_init_func =&usermap_test_zone_init;
     init_zones[0] = "start_zone";
     level thread zm_zonemgr::manage_zones( init_zones );

     level.pathdist_type = PATHDIST_ORIGINAL;

     //Starting weapon
     startingWeapon = "smg_ppsh";
     weapon = getWeapon(startingWeapon);
     level.start_weapon = (weapon);
    }

  5. You can find a list of weapons by going to Steam\steamapps\common\Call of Duty Black Ops III\share\raw\gamedata\weapons\zm
  6. Open the zm_levelcommon_weapons.csv file in excel to view the list of weapon names. You can also open the file in a text editor and look at the first name in the comma seperated list on each line.
