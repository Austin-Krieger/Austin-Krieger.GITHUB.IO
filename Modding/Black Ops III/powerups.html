Additional Powerups

This tutorial includes zombie blood, fast feet, zombie cash, time warp, and infinite ammo.

  1. Here are the folders you will need to open for this tutorial, starting with our root folder D:\Steam\steamapps\common\Call of Duty Black Ops III\

    2. usermaps\zm_tutorial_level\scripts\zm

    usermaps\zm_tutorial_level\zone_source

    usermaps\zm_tutorial_level\sound\zoneconfig

    share\raw\sound\aliases

  2. Download: Powerups.zip
  3. Once you extract the files, open up the Powerups folder and drag the contents into the root folder. You can also move the actual files into their appropriate folders yourself.
  4. In your scripts\zm folder, open the zm_tutorial_level.gsc file.
  5. Add the following lines below the line: #using scripts\zm\zm_usermap;

    6. // Powerups
    #using scripts\powerups\powerup_time_warp;
    #using scripts\powerups\powerup_fast_feet;
    #using scripts\powerups\powerup_bottomless_clip;
    #using scripts\powerups\powerup_money;
    #using scripts\powerups\powerup_zombie_blood;

  6. In your Maps' zone_source folder, open the zm_tutorial_level.zone file.
  7. Add the following lines to the end of the file:

    8. // Powerups
    scriptparsetree,scripts/powerups/powerup_time_warp.gsc
    scriptparsetree,scripts/powerups/powerup_time_warp.csc
    xmodel,hour_glass
    material,time_warp
    scriptparsetree,scripts/powerups/powerup_fast_feet.gsc
    scriptparsetree,scripts/powerups/powerup_fast_feet.csc
    xmodel,zombietron_boots
    material,fast_feet
    scriptparsetree,scripts/powerups/powerup_bottomless_clip.gsc
    scriptparsetree,scripts/powerups/powerup_bottomless_clip.csc
    xmodel,bottomless_clip
    material,bottomless_clip
    scriptparsetree,scripts/powerups/powerup_money.gsc
    scriptparsetree,scripts/powerups/powerup_money.csc
    xmodel,powerup_money
    scriptparsetree,scripts/powerups/powerup_zombie_blood.gsc
    scriptparsetree,scripts/powerups/powerup_zombie_blood.csc
    xmodel,zombie_blood
    material,zombie_blood

  8. From the root folder, go to your share\raw\sound\aliases folder and move the zmb_powerups.csv file you downloaded earlier into this folder.
  9. Next go to your Maps' sound\zoneconfig folder and open up the zm_tutorial_level.szc file.
  10. Add in the new alias below user_aliases as shown here:

    11. {
    "Type" : "ALIAS",
    "Name" : "user_aliases",
    "Filename" : "user_aliases.csv",
    "Specs" : [ ]
    },
    {
    "Type" : "ALIAS",
    "Name" : "zmb_powerups",
    "Filename" : "zmb_powerups.csv",
    "Specs" : [ ]
    },

  11. You should now have additional powerups in your map. If you change folder names or anything like that you will need to make changes to the files we opened as well as the .gdt files in our model_export folder.
