Increasing the Player Starting Points

  1. Open your Map folder and go to the scripts\zm folder. Look for the file named zm_tutorial_level.gsc

    2. D:\Steam\steamapps\common\Call of Duty Black Ops III\usermaps\zm_tutorial_level\scripts\zm

  2. Right Click the file and edit it with your preferred text editor.
  3. Add the following line near the top of the Main() function:

    4. level.player_starting_points = 500000;


    function main()
    {
     zm_usermap::main();

     // Perk Limit
     level.perk_purchase_limit = 20;

     // Starting Points
     level.player_starting_points = 500000;

     level._zombie_custom_add_weapons =&custom_add_weapons;

     // Setup the levels Zombie Zone Volumes
     level.zones = [];
     level.zone_manager_init_func =&usermap_test_zone_init;
     init_zones[0] = "start_zone";
     level thread zm_zonemgr::manage_zones( init_zones );

     level.pathdist_type = PATHDIST_ORIGINAL;
    }

 

Change Player Starting Weapon

This is meant to change your starting pistol when loading into the map.

  1. Open your Map folder and go to the scripts\zm folder. Look for the file named zm_tutorial_level.gsc

    2. D:\Steam\steamapps\common\Call of Duty Black Ops III\usermaps\zm_tutorial_level\scripts\zm

  2. Right Click the file and edit it with your preferred text editor.
  3. We need to add a few lines of code to the bottom of the main() function as shown here:

    4. function main()
    {
     zm_usermap::main();

     // Perk Limit
     level.perk_purchase_limit = 20;

     // Starting Points
     level.player_starting_points = 500000;

     level._zombie_custom_add_weapons =&custom_add_weapons;

     // Setup the levels Zombie Zone Volumes
     level.zones = [];
     level.zone_manager_init_func =&usermap_test_zone_init;
     init_zones[0] = "start_zone";
     level thread zm_zonemgr::manage_zones( init_zones );

     level.pathdist_type = PATHDIST_ORIGINAL;

     // Starting Weapon
     startingWeapon = "smg_ppsh";
     weapon = getWeapon(startingWeapon);
     level.start_weapon = (weapon);
    }

  4. You can find a list of weapons in the share\raw\gamedata\weapons\zm folder:

    5. Steam\steamapps\common\Call of Duty Black Ops III\share\raw\gamedata\weapons\zm

  5. Open the zm_levelcommon_weapons.csv file in excel or a text editor to view the list of weapons. In a text editor you will see a comma seperated list on each line.
