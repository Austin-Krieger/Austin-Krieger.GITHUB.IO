Adding an Ascension Style Black & White Vision Filter

  1. Here are the folders you will need to open for this tutorial, starting with our root folder D:\Steam\steamapps\common\Call of Duty Black Ops III\

    2. usermaps\zm_tutorial_level\scripts\zm

    usermaps\zm_tutorial_level\zone_source

  2. Open your Maps' .gsc file from the usermaps\zm_tutorial_level\scripts\zm folder.
  3. Inside the main() function add the following lines:

    4. // Black & White Vision
    thread bw_vision();

  4. Scroll up to the line #using scripts\zm\zm_usermap; and paste the following lines after it:

    5. // Black & White Vision
    #using scripts\zm\_zm_laststand;
    #using scripts\shared\lui_shared; // flash

  5. Now scroll down to the very bottom and paste the following. Feel free to play with the values.

    6. // Black & White Vision
    function bw_vision()
    {
     level flag::wait_till( "initial_blackscreen_passed" );
     wait(2);

     VisionSetNaked( "zombie_last_stand", 2 );

     thread ending_bw_vision();
    }

    function ending_bw_vision()
    {
     level flag::wait_till( "power_on" );
     lui::screen_flash( 0.2, 0.5, 1.0, 0.8, "white" ); // flash
     wait(0.5);
     VisionSetNaked( "zm_factory", 0.5 );
    }

  6. Now we need to open our Maps' zone file zm_tutorial_level.zone from the zone_source folder and add the following lines at the end:

    7. // Black & White Vision
    rawfile,vision/zombie_last_stand.vision
    rawfile,vision/zm_factory.vision
