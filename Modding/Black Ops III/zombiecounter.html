Zombie Counter

Following this tutorial will give you a remaining zombies count in the top left of your screen for each round.

  1. Here are the folders you will need to open for this tutorial, starting with our root folder D:\Steam\steamapps\common\Call of Duty Black Ops III\

    2. usermaps\zm_tutorial_level\scripts\zm

  2. Open your maps .gsc file zm_tutorial_level.gsc and add the following lines to the main() function:

    3. // Zombie Counter
    _INIT_ZCOUNTER();

  3. Download this text file: ZombieCounterCode.txt
  4. Open up the ZombieCounterCode.txt file and copy all of the text inside.
  5. Back in the zm_tutorial_level.gsc file scroll to the bottom and paste the copied code.
  6. There are customizable parts to this code, but I won't be covering that right now.
