Adding Wall Buys

  1. In the Entity Browser find Misc, Right Click, and Toggle Favorite.
  2. Right Click in the Game View and hover over Misc > prefab. Without favoriting Misc this option won't appear.
  3. A new window will pop up, double click zm > zm_core.
  4. Search for spawnable_weapon_bowie.map and hit Open to add the Bowie Knife to the map.
  5. Move the prefab up against the wall and if it's backwards like in the image below, press R to switch to rotation handles and use the Z axis to flip it around. You can also use Shift + D to rotate an entity in 90° increments.
  6. We can change the rotation manually from the Entity Info Pane. This is a better way to make sure our item is rotated 180°.
  7. When moving the prefab around you will notice that it snaps to the corners of the grid squares. We need to increase the number of grid squares to move it more precisely. Find the button that looks like a grid and choosing Grid 8.
  8. We need to line up the prefab with the floor & wall as shown below. Remember Ctrl + Tab will let you change the current Grid View. We want the wall buy to be very slightly off the wall, so if you switch the grid size to Grid 1 we can add a tiny square of space between the knife and the wall.
  9. The Bowie Knife should now be added and look similar to this in the Game View.

Changing Weapon Values

You can change the prices of your weapons as well as their attachments and ammo costs.

  1. Open your Map folder and create this folder structure gamedata > weapons > zm.

    2. usermaps\zm_tutorial_level\gamedata\weapons\zm

  2. You can then find the weapons list inside the share\raw\gamedata\weapons\zm folder:

    3. share\raw\gamedata\weapons\zm

  3. Copy the zm_levelcommon_weapons.csv file and paste it into the zm folder you just made.
  4. Next we will open our maps' zone file: zm_tutorial_level.zone

    5. usermaps\zm_tutorial_level\zone_source

  5. Add the following lines to the bottom of the file:


    6. // Updated Wall Weapon Costs
    stringtable,gamedata/weapons/zm/zm_levelcommon_weapons.csv

  6. Finally we can open the zm_levelcommon_weapons.csv file in excel or a text editor to view the list of weapons and make our changes.

Changing Bowie Knife Cost

The Bowie Knife is a little more difficult to change the price on, so we have a few steps to do in order to change it.

  1. First we need to find the _zm_weap_bowie.gsc file in the share\raw\scripts\zm folder:

    2. share\raw\scripts\zm

  2. Copy this file and go to your maps' scripts\zm folder:

    3. usermaps\zm_tutorial_level\scripts\zm

  3. Paste the file and open it with your text editor.
  4. You will see cost = 3000; change this to whatever value you want.

    5. function private __main__()
    {
     if( isdefined( level.bowie_cost ) )
     {
      cost = level.bowie_cost;
     }
     else
     {
      cost = 3000;
     }
    ...

  5. Next we need to open the maps' zone file zm_tutorial_level.zone:

    6. usermaps\zm_tutorial_level\zone_source

  6. Add the following lines to the bottom of the file:

    7. // Update Bowie Knife Cost
    scriptparsetree,scripts\zm\_zm_weap_bowie.gsc

  7. This will override the original script file and update the cost of the bowie knife wall buy.
