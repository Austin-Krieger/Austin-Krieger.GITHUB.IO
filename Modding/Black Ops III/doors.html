Double Doors

Models

  1. In the Models Pane (M) search for p7_door_metal_industrial_02_left_white and drag it onto either the Map or Grid view.
  2. Do the same with p7_door_metal_industrial_02_right_white.
  3. Line them up with the floor and each other, like so:

Clip

  1. Select clip from the Textures Pane and make a clip the size of the doors themselves so that players can't walk through them.
  2. Next we need to convert our clip to a brushmodel from the Entity Browser Pane. Look for Script > brushmodel and Double Click.
  3. With our new brushmodel selected, go to the Entity Info Pane (N) and change the targetname to test_door.

    4. targetname: test_door

  4. Click Add KVP

    5. Property / Key: script_noteworthy
    Value: clip

Trigger

  1. We need a trigger box in front of our doors so that players will get a prompt to purchase it when they approach.
  2. Option #1: make a clip in front of the doors. Go to the Entity Browser Pane and look for Trigger > use and with our new clip still selected, Double Click to convert to a trigger.
  3. Option #2: drag Trigger > use onto the Grid or Game view and resize it the same way you would the clip from Option #1.
  4. With our new Trigger selected, go to the Entity Info Pane (N) and change the targetname to zombie_door.

    5. targetname: zombie_door

  5. We also need to set the cost of our door. Hit Add KVP and set the price of the door (Default is 1000).

    6. Property / Key: zombie_cost
    Value: 750

  6. Click Add KVP

    7. Property / Key: target
    Value: test_door

  7. Click Add KVP (This one is used when you have multiple zones)

    8. Property / Key: script_flag
    Value: enter_test_zone

Doors

  1. It may be a struggle to select a specific entity by this point, so we can press H to hide a selected entity and Shift + H to show everything you have hidden. Shift + Ctrl + H will reveal only the last item that was hidden.
  2. Select a Door and return to the Entity Browser Pane. Look for Script > model and Double Click. Do the same for the other door and continue to the next step.
  3. With a Door model selected go to the Entity Info Pane (N) and scroll down to targetname. We can change this to whatever as long as it matches throughout the tutorial.

    4. targetname: test_door

  4. Click Add KVP

    5. Property / Key: script_string
    Value: rotate

  5. Rotate the door in the Game view until you find a rotation that fits your needs. For the door on our right: {0, 120, 0} looks good.
  6. Click Add KVP

    7. Property / Key: script_angles
    Value: 0 120 0

  7. Follow the same steps for the door on the left. This time {0, 240, 0} will work.

    8. Property / Key: script_angles
    Value: 0 240 0

 

Electric Door

Here we will make a door that opens when the power is turned on.

Door

  1. Select clip from the Textures Pane.
  2. Make a clip the size of the electric door you want to make.
  3. Back in the Textures Pane search for t7_metal_garage_door_rollup_01_bare and Double Click.
  4. With the new clip selected, go to the Entity Browser and find Script > brushmodel and Double Click.
  5. With our door selected, go to the Entity Info Pane (N) and change the targetname to test_garage_door.

    6. targetname: test_garage_door

  6. Click Add KVP

    7. Property / Key: script_string
    Value: move

  7. Click Add KVP (In this example the door will move straight up to 100 on the Z axis)

    8. Property / Key: script_vector
    Value: 0 0 100

Clip

  1. Create another clip the size of your door.
  2. With the new clip selected, go to the Entity Browser and find Script > brushmodel and Double Click.
  3. With our clip selected, go to the Entity Info Pane (N) and change the targetname to test_garage_door.

    4. targetname: test_garage_door

  4. Click Add KVP

    5. Property / Key: script_noteworthy
    Value: clip

Trigger

  1. Add one last clip in front of the door.
  2. With the new clip selected go to the Entity Browser and find Trigger > use and Double Click.
  3. With our new Trigger selected, go to the Entity Info Pane (N) and change the targetname to zombie_door.

    4. targetname: zombie_door

  4. Click Add KVP

    5. Property / Key: script_noteworthy
    Value: electric_door

  5. Click Add KVP

    6. Property / Key: target
    Value: test_garage_door

 

Debris Door

Here we will make a debris door out of whatever models we want.

Models

  1. In the Models Pane search for p7_cai_stacking_cargo_crate and add a few to the map.
  2. Select each crate individually and go to the Entity Browser Pane, find Script > model and Double Click.
  3. For each of our Script > models, go to the Entity Info Pane (N) and change the targetname to test_debris.

    4. targetname: test_debris

  4. Click Add KVP (On each).

    5. Property / Key: target
    Value: test_struct

Struct

  1. In the Entity Browser Pane find Script > struct and drag it onto the map.
  2. Since this is where the debris will fly off to, we should place it high above our debris.
  3. With our struct selected, go to the Entity Info Pane (N) and change the targetname to test_struct.

    4. targetname: test_struct

Clip

  1. Draw a clip around the crates.
  2. In the Entity Browser Pane go to Script > brushmodel and Double Click.
  3. With our new brushmodel selected, go to the Entity Info Pane (N) and change the targetname to test_debris.

    4. targetname: test_debris

  4. Click Add KVP

    5. Property / Key: script_noteworthy
    Value: clip

Trigger

  1. Add another clip in front of our debris.
  2. In the Entity Browser Pane go to Trigger > use and Double Click.
  3. With our new Trigger selected, go to the Entity Info Pane (N) and change the targetname to zombie_debris.

    4. targetname: zombie_debris

  4. Click Add KVP

    5. Property / Key: zombie_cost
    Value: 1250

  5. Click Add KVP

    6. Property / Key: target
    Value: test_debris

 

Making a Brushmodel Door

  1. Here are the folders you will need to open for this tutorial, starting with our root folder D:\Steam\steamapps\common\Call of Duty Black Ops III\

    2. map_source\_prefabs\Doors

  2. Go to your _prefabs folder and create a new folder named Doors or place it wherever you want so long as you can find it later.
  3. Download: brushmodel_door.map
  4. You can also perform the following steps to make your own.
  5. Select clip from the Textures Pane and create a clip roughly the size of a door.
  6. Make a copy of the clip and line it up on the side of the door where the hinges would be.
  7. Select the clip that will act as the door and return to the Textures Pane. Search for t7_decal_fixture_door_wood_01 and double click.
  8. Open the Surface Inspector (S) and in the fit section select both.
  9. Select the other clip and Right Click > Make Non-Colliding. This will make it so the player will not run into an invisible door.
  10. Next, select both clips Right Click > Script > brushmodel or go to the Entity Browser and double click Script > brushmodel.
  11. With both clips still selected Right Click > Entity > Add Selected Brushes. This will group the two clips.
  12. The next steps you can do with only one clip selected.
  13. Change the targetname to brushmodel_door and if you did the previous steps correctly, the targetname of our other clip will be the same. This applies to the KVPs we will be adding as well.

    14. targetname: brushmodel_door

  14. On either of the clips click Add KVP

    15. Property / Key: script_string
    Value: rotate

  15. Click Add KVP

    16. Property / Key: script_angles
    Value: 0 -120 0

  16. You can check the angle by selecting both clips and rotating them (R). Look at the bottom of the mod tools window and you will see the z value changing. Use that value for the Y value of our script_angles KVP. Doesn't make sense, but it works.
  17. Now we can make the door buyable.
  18. Make a clip in front of the door and Right Click > Trigger > use or go to the Entity Browser and double click on Trigger > use.
  19. Change the targetname to zombie_door.

    20. targetname: zombie_door

  20. Click Add KVP

    21. Property / Key: zombie_cost
    Value: 750

  21. Click Add KVP

    22. Property / Key: target
    Value: brushmodel_door

  22. If you want it to also unlock the next zone we will need to add another KVP. Change the value to the name of the zone you want to enter.

    23. Property / Key: script_flag
    Value: enter_test_zone

  23. You now have a working door made of brushmodels.
  24. If you downloaded the prefab above, open your map in Radiant and Right Click > Misc > prefab to place the door and stamp the prefab or Right Click > Prefab > Enter Prefab, and make your changes.
