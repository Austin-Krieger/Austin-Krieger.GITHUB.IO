Getting Started

×

Install DSMapStudio

In order to merge mods (regulation.bin files) and make your own changes to the game, you will want to get DSMapStudio . I do not recommend Yapped Rune Bear as it is outdated at the moment and will make merging mods a lot more difficult.

  1. Download and install the latest release. You can keep the program wherever you want, I just keep it in the downloads folder for example.
  2. At this point you will have some issues running the program due to missing runtimes.
  3. The System Requirements section of their GitHub page will provide more details regarding this issue, but for now all you need to do is install these two runtimes and the program should work fine.
  4. Microsoft .NET Core 6.0 Desktop Runtime: https://aka.ms/dotnet/6.0/windowsdesktop-runtime-win-x64.exe .
  5. Microsoft .NET Core 6.0 ASP.NET Core Runtime: https://aka.ms/dotnet/6.0/aspnetcore-runtime-win-x64.exe .

Install Yabber

When merging animations and sfx changes for your latest weapon moveset mods, you will need a program like yabber to unpack and repack the necessary .dcx files

  1. There is no need to run the executable (yabber.exe) just unzip the folder you downloaded and open up the lib folder. We need to copy an Elden Ring file to this location.
  2. In a new file explorer window, navigate to Steam > steamapps > common > Elden Ring > Game and copy this file: oo2core_6_win64.dll
  3. Paste the file in your lib folder and Yabber is ready to go.

Install UXM Selective Unpacker

To be able to fully utilize the features of DSMapStudio you may want to unpack the Elden Ring executable. For this you will need to get UXM Selective Unpacker from NexusMods.

  1. The Unpack button will extract all files from the game archives. Any files already preset will be skipped.
  2. The Patch button will modify the exectuable to use those extracted files instead of looking in the archives. Most users will not be using this button and will only unpack the files they need. Patching the executable without also unpacking all files from the archives will also crash the game.
  3. The Restore button will restore the original executable and delete the extracted files.
  4. The Abort button will cancel any operation in progress.
  5. We will only need to use the Unpack and Restore buttons.
  6. Go ahead and Unpack the game whenever you are ready. You do not have to worry about restoring the files until you are completely done modifying and testing your mods in game or have no use for the full functionality of DSMapStudio or DSAnimStudio. Just to be clear, Restore only removes (deletes) the files and folders that were "unpacked" and has no effect on the executable or mods we will be working with.

Install DSAnimStudio

Another program that can be useful for both merging and creating mods is: DSAnimStudio .

  1. More instructions can be found in Basic User Instructions on their GitHub page.
  2. You can use this program to open .dcx files such as: c0000.anibnd.dcx and view the animations inside.

Running Anti-cheat Toggler

Before running the game with mods, we need to install and run the Anti-cheat Toggler and make sure Easy Anti-cheat is disabled.

  1. Extract the files to the Steam > steamapps > common > Elden Ring > Game folder.
  2. Run toggle_anti_cheat.exe and a message will pop up notifying you that Anti-cheat has been disabled.
  3. Next you could install your mods and run start_game_in_offline_mode.exe, but I will be installing Seamless Co-op so I can mod the game and still play with friends who have the same mods.

Install Elden Mod Loader

Before we get started installing and merging mods for Seamless Co-op we may want Elden Mod Loader from Nexus Mods for choosing some quality of life mods and their load order. Move the downloaded files to the Elden Ring > Game folder.

  1. The two files downloaded need to be placed inside of the Elden Ring > Game folder.
  2. Next we need to install Mod Engine 2.0.0.

Install Mod Engine 2.0.0

We will be using Mod Engine to load our mods and start up the game.

  1. At the time of this tutorial being written, Mod Engine 2.0.0 was on version "preview4".
  2. Whenever you want to play the game, you will now run: launchmod_eldenring.bat. This will run any .dll files in your "mods" folder which is located in the Elden Ring > Game folder as well as your own merged mods located in the Mod Engine 2.0.0 > mod folder.

Install Seamless Co-op

Finally we can get Seamless Co-op from Nexus Mods and start playing with friends.

  1. Extract the files and place the .dll and .ini files inside of the Elden Ring > Game > mods folder we added when we got Elden Mod Loader.
  2. With the files placed inside of the mods folder they will automatically be called when we start the game from our Mod Engine 2.0.0 file mentioned previously.
  3. Another mod we can place inside of the mods folder is the Skip The Intro mod which will skip the bright white screan that blinds you when launching the game.
  4. Lastly, you will need to set a password for Seamless Co-op which will allow you to connect with your friends. Open the seamlesscoopsettings.ini file and choose a password for everyone to save after this equals sign: cooppassword =
  5. Now we can move on to the next page: Merging Mods where we will discuss how to merge the mods we have downloaded.
