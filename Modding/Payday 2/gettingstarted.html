Getting Started

×

Install SuperBLT

The original installation guide can be found here .

  1. Download and install the Microsoft Visual C++ 2017 Redistributable package (x86) .
  2. Download the Latest Release DLL , and place it in your PAYDAY 2 folder (D:\Steam\steamapps\common\PAYDAY 2) alongside payday2_win32_release.exe.
  3. You can also get to your PAYDAY 2 files by Right Clicking the game in your Steam library > Properties... > LOCAL FILES > Browse...
  4. Start the game, and SuperBLT will prompt you to download the basemod. Select Yes, and it will notify you when it's done downloading. At this point, SuperBLT has been fully installed.
  5. Run the game again to check that it has installed.

Install BeardLib

  1. First you need to create a mods folder within the PAYDAY 2 folder.
  2. You can download it from modworkshop .
  3. Extract the .zip file and move it to your newly created mods folder.
  4. Once you have your mod files in your PAYDAY 2 local files, run PAYDAY 2 with your keyboard and mouse (controller won't let you select the updates available prompt).
  5. Update your mods.

Install WolfHUD

  1. Download the files from GitHub
  2. Extract the folder and place the files in your mods folder.
  3. Start the game again and you will be prompted to update Federal Inventory.
