Installing Software

In order to edit the L4D2 files, you will need to download and install a couple of free programs which I have provided below.

  1. Download GCFScape: gcfscape186.exe
  2. Download VTFEdit: vtfedit125-11.exe

Extracting Original L4D2 Files

  1. Locate your Steam folder and continue to the second Left 4 Dead 2 folder.

    2. D:\Steam\steamapps\common\Left 4 Dead 2\left4dead2

  2. Open up GCFScape and click File > Open
  3. Locate the folder from the previous step and click on the pak01_dir.vpk file.
  4. Click Open.
  5. Your program should now look like this:
  6. Navigate to the file you wish to edit. The flashlight texture for example.
  7. Click on the file and drag it out of the program and into a folder of your choice. This will copy the file to your folder.

Converting The File

  1. This time we will need to open VTFEdit.
  2. Click File > Open and locate the file you copied for editing.
  3. Once you have opened the file you will get a preview of the file and some additional information:
  4. Take note of the Flags that the file uses for when you import your own. The flashlight file uses the Flags: Clamp S, Clamp T, No Level Of Detail, and Clamp All.
  5. Next you will need to export the file as a targa file. Click File > Export and save the file as FileName.tga
  6. This file you will be able to open and edit in another program such as Photoshop or Illustrator.
  7. With Photoshop open, click on File > Open and select the .tga file we just made. It should look like this:
  8. Make your changes and save the file as a .tga again. If you are using a newer version of Photoshop then you will need to use File > Save a Copy to save it as a .tga
  9. Next you will need to open VTFEdit again. Select File > Import and select the .tga file you just created.
  10. A new window will pop up with some Import settings. For the flashlight001 file we need to make sure that its max size is 512x512 and the format is BGR888.
  11. Apply the Flags you took note of earlier.
  12. Now we need to convert the file back to a .vtf file format.
  13. Select File > Save As... and make sure the name matches with the name of the original file.

Packaging Your Files

  1. To package your mods and even upload to the workshop, you will first need to install the Left 4 Dead 2 Authoring Tools. Search through the uncategorized/tools section of Steam and install the program.
  2. The new executable file that we need can be found in your bin folder:

    3. D:\Steam\steamapps\common\Left 4 Dead 2\bin

  3. You are looking for vpk.exe
  4. Right Click on the executable and click Create Shortcut.
  5. Move this shortcut to your modding folder for later convenience.
  6. Once you have your mod files in a parent folder you will be able to drag said parent folder onto the vpk shortcut. This will package the folder and its containing files into a new readable mod file for the game.
  7. To test your mods locally you will need to find your addons folder.

    8. D:\Steam\steamapps\common\Left 4 Dead 2\left4dead2\addons

  8. Drag your packaged file to this folder and restart your game if it's already open.
  9. Mods installed from the workshop will show up in your addons/workshop folder.
