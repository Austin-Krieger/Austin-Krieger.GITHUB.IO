Adding Custom Stencils

  1. This tutorial will walk you through making a custom transparent stencil for Left 4 Dead 2.
  2. First thing we need is a .png file. You can find an image online or even convert a .jpg file in a photo editting program such as Photoshop.
  3. The next thing we need to do is resize the photo to an acceptable size such as: 512x512
  4. Open up Photoshop and make a new file that is 512x512 pixels.
  5. Place and resize your image on the canvas. You may have a very specific issue later involving the coach stencil appearing with new colors, in which case you can try deleting any hidden layers you have and trying again.
  6. Click File > Export As.. and change the Format to PNG and Image Size to 512x512.
  7. Click Export and save the file with the name of your choice. There isn't a required naming convension for these files.
  8. Now that we have the .png file we need to convert it to an acceptable file format for Left 4 Dead 2.
  9. You can use VTFEdit to convert files, but for stencils we have a more simplified program we can use called spraygen-134: https://gamebanana.com/tools/download/5230
  10. After downloading and extracting the program files we can open the program: spraygen.exe
  11. Click on the Import button and locate the .png file we made earlier. The size should default to 512x512.
  12. Click on the Transparent background checkbox and hit Convert to VTF.
  13. Save your file.
  14. Next we need to locate the the logos folder inside of the Left 4 Dead 2 folder. This is where you will move your custom stencils (Recommended).

    15. D:\Steam\steamapps\common\Left 4 Dead 2\left4dead2\materials\vgui\logos

  15. Now we can finally open the file inside of Left 4 Dead 2.
  16. Start up the game and go to OPTIONS > Multiplayer.
  17. Click IMPORT SPTRAY...
  18. Navigate to the materials/vgui/logos folder and open the .vtf file we made using spraygen-134
  19. Your image should appear in the middle; make sure to click DONE.
  20. Load into a mission and press T to paint your image onto a wall.
