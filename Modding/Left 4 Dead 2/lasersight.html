Laser Sight Color Change

  1. Locate your Steam folder and continue to the second Left 4 Dead 2 folder.

    2. D:\Steam\steamapps\common\Left 4 Dead 2\left4dead2

  2. Open up GCFScape and click File > Open
  3. Locate the folder from the previous step and click on the pak01_dir.vpk file.
  4. Click Open.
  5. Scroll down and open the particles folder.
  6. Look for the file named weapon_fx.pcf and as with the other tutorials, click on and then drag this file into a folder of your choice. Make sure to copy the folder structure containing this file and make an addoninfo.txt file as well.
  7. Now that we have the file we need we can move on to how we can open the Hammer Particle Editor Tool for Left 4 Dead 2.
  8. In order to get to this tool we need to first locate and open the Left 4 Dead 2 Authoring Tool shown below. If you do not already have the program installed, you will need to look in your Steam Library and inside the Tools section.
  9. Now that we have the Authoring Tool open we need to double-click on Left 4 Dead 2 (Tools Mode) to open the editor.
  10. Next we need to open up the Particle Editor Tool.
  11. Click on the Tools button at the top and select the Particle Editor. You won't see a big change in the current window, but this will allow us to open the file we copied earlier.
  12. Click File > Open and locate the weapon_fx.pcf file and hit Open.
  13. Scroll down a little ways and look for weapon_laser_sight. There will be a total of 6 effects with similar names that we will want to apply our color changes to.
  14. Once you have the HEX code for the color you want, we will need to convert it to HSV values using an online converter like this one: https://colordesigner.io/convert/hextohsv
  15. RGBA values are also acceptable.
  16. Each of the 6 effects has a color1 and color2 value for us to change.
  17. Once you are done making all of these changes, you can use File > Save As... and replace the original file in your mod folder.
  18. If you have done everything correctly, you can package up and place the mod in your addons folder like usual.
  19. To quickly test our new mod, check out the console commands page where you can learn how to spawn in a laser sight box anywhere you want.
