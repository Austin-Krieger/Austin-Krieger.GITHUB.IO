Applying a Flashlight Filter

×

Flashlight Filters

  1. Locate your Steam folder and continue to the second Left 4 Dead 2 folder.

    2. D:\Steam\steamapps\common\Left 4 Dead 2\left4dead2

  2. Open up GCFScape and click File > Open
  3. Locate the folder from the previous step and click on the pak01_dir.vpk file.
  4. Click Open.
  5. Your program should now look like this:
  6. Navigate to the material > effects folder and locate the flashlight001.vtf file.
  7. Click on the file and drag it out of the program and into a folder of your choice. This will copy the file to your folder.
  8. Note: If you look inside the materials > particle folder, you will see a beam_flashlight.vtf file. It couldn't hurt to apply the same changes discussed in this tutorial to this file as well. Make sure you take note of the flags and size information of the original file before continueing on with your own. Flags: No Mipmap Size: 64x256.

Converting The File

  1. This time we will need to open VTFEdit.
  2. Click File > Open and locate the file you copied for editing.
  3. Once you have opened the file you will get a preview of the file and some additional information:
  4. Take note of the Flags that the file uses for when you import your own. The flashlight file uses the Flags: Clamp S, Clamp T, No Level Of Detail, and Clamp All.
  5. Next you will need to export the file as a targa file. Click File > Export and save the file as flashlight001.tga
  6. This file you will be able to open and edit in another program such as Photoshop or Illustrator.
  7. For a more detailed walkthrough on editing the flashlight in photoshop, scroll down to the next section.
  8. With Photoshop open, click on File > Open and select the .tga file we just made. You should now see this:
  9. Make your changes and save the file as a .tga again. If you are using a newer version of Photoshop then you will need to use File > Save a Copy to save it as a .tga
  10. Next you will need to open VTFEdit again. Select File > Import and select the .tga file you just created.
  11. A new window will pop up with some Import settings. For the flashlight001 file we need to make sure that its max size is 512x512 and the format is BGR888.
  12. Apply the Flags you took note of earlier.
  13. Now we need to convert the file back to a .vtf file format.
  14. Select File > Save As... and make sure the name matches with the name of the original file (flashlight001.vtf).
  15. Now that we have our new flashlight file, we need to copy the folder structure of where the file originally was.
  16. Start by making a new folder by the name of your mod.
  17. Inside this folder add a folder named materials.
  18. Within the materials folder, make a folder named effects.
  19. Place your flashlight001.vtf file inside.
  20. One last thing we should have before packaging our mod, is an AddonInfo file. This is not necessary for making your mod work, but it's best to use one if you want a name and description in game, especially if you plan to publish the mod to the workshop.
  21. Go to the folder for your mod and make a new .txt file named addoninfo.txt
  22. Open the file and use the screenshot below as an example. Note that there are character length restrictions.
  23. Select the folder containing your mod and package it as I describe in the Getting Started page.
  24. Remember to move the packaged file to the addons folder, so you can test it locally.

Photoshop Walkthrough

  1. Start by copying and then hiding the background layer.
  2. Select the Elliptical Marquee Tool and make a large selection. Hold Shift to keep the circle nice and round.
  3. Next you will need to add a Hue/Saturation Layer Mask by clicking on the half filled in circle in the bottom right of the program window.
  4. If you have a HEX code for the color you want, then you can go to this site to convert it to an HSL value: https://htmlcolors.com/hex-to-hsl
  5. Check the colorize box and then put in your Hue/Saturation/Lightness values for the color you want.
  6. Next you will need to add a Blending Mode to the layer. I suggest either Overlay or Multiply.
  7. With the box on the right next to the layer name selected you will see the properties of the layer mask selection and be able to make changes to the density and feathering if you want. I made a quick change to these values, but that was just an arbitrary change for the sake of this example.
  8. You can make more drastic changes to the flashlight from here, but the goal of this tutorial was just to change the color.
