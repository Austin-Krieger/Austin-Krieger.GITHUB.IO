Enabling Console Commands

  1. To use console commands, we need to enable the Developer Console from the main menu.
  2. After opening the game to the main menu, go to OPTIONS > Keyboard/Mouse
  3. You will see the option ALLOW DEVELOPER CONSOLE. Click on it and select Enabled.
  4. Click Done.
  5. To use the cheats we will need to load into a map using the console.
  6. From the main menu press the tilde key (~) and the console will open.
  7. Type in sv_cheats 1 to enable cheats.
  8. If you now type map and hit space you will see a list of campaign chapters to load into. Using map c1m1_hotel will load you into the first chapter of Dead Center.
  9. Pressing the escape key in game will pause the game and show the Developer Console.
