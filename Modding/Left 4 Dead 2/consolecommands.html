Console Commands

  1. Enable the developer console in the main menu
  2. Press the tilde key (~) to open the console
  3. Type in sv_cheats 1 to enable cheats
  4. Type map and hit space to see a list of campaign chapters to load into

    5. Command Description
    sv_cheats 1 Enable Cheats
    god 1 Enable God Mode
