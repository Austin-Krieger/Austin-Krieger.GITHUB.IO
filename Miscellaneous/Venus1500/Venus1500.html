Installing the Correct Driver

  1. You may need to download the correct drivers for your device before you can progress (DUB-E100_B1_Driver_3.4.0) is the driver I recommend.
  2. I downloaded the B1 driver here: https://support.dlink.com/ProductInfo.aspx?m=DUB-E100
  3. Select the hardware revision B and driver version Driver (3.40) and hit download
  4. Extract the zip file and install the driver
  5. Once the driver is installed, your device should be working properly as shown below.

Connecting your Venus1500 Software with your Daktronics Sign

  1. If you have the Ethernet Bridge Radio Diagnostics software installed it probably looks like this:
  2. "No Radios Found" does not mean you cannot connect to your sign.
  3. For your adapter and your sign to communicate, they cannot have the same IP Address.
  4. Open the Control Panel by first tapping the Windows key on your keyboard or clicking on the Windows logo in the bottom left of your screen. Type Control Panel and a suggested app should appear, click on it or hit enter.
  5. Once the program has loaded up, click on the Network and Internet link.
  6. Now go to the Network and Sharing Center and click on Change adapter settings
  7. A new window will appear with a list of Network Connections, one of them will say D-Link DUB-E100 USB 2.0 Fast Eth... like so:
  8. Right click this connection and click on Properties.
  9. Click yes if you get a prompt and a new window will appear.
  10. Click on Internet Protocol Version 4 (TCP/IPv4) and click on the Properties button below.
  11. Click yes if you get another prompt and a new window will once again appear.
  12. We need to change the IP address to something different from the address we gave our display in the Venus1500 software (172.16.192.25) to (172.16.192.20) as shown in the previous image.
  13. Hit the OK button and return to the Venus1500 software.
  14. You should now seee a Display Status bar filled mainly in green on the left.
  15. If you don't see this, make sure your display is selected in the middle drop down box and it couldn't hurt to restart the program.
 

Transfering Presentations from an Old Computer

  1. To Export your previously created presentations from an old computer, navigate to your Daktronics folder in your Program Files (x86) folder. Continue into the Nucleus Server and finally the Database Utility Folder. C:\Program Files (x86)\Daktronics\Nucleus Server\Database Utility
  2. Run the application named Nucleus Database Utility.
  3. Once the program loads up you will see the Save/Export option which can be used when moving presentations from an old computer. Export your presentations to a .nucbak file with a name of your choosing and save said file to a USB.
  4. Transfer your USB to the new computer and transfer the file to your Desktop (Or wherever you will be able to find it).
  5. Navigate to your Daktronics folder once again and Run the Nucleus Database Utility. C:\Program Files (x86)\Daktronics\Nucleus Server\Database Utility
  6. This time select the Load/Import option. This option is used when adding your presentations to the new computer.
  7. Returning to the Venus1500 software and clicking on the Content tab you should see your list of presentations. For example:
  8. The last thing to do is to add some presentations to your schedule in the Schedule tab, Publish To Display and confirm that the sign has updated and is working properly.
